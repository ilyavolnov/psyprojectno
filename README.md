# PsyProject

Сайт психолога Маргариты Румянцевой с админ-панелью и Telegram-ботом

## Описание

Это веб-приложение для сайта практикующего психолога, включающее:
- Публичный сайт с курсами и вебинарами
- Админ-панель для управления контентом
- Telegram-бот для автоматизации взаимодействия с клиентами
- Систему заявок и управления заказами

## Структура проекта

- `admin/` - файлы админ-панели
- `backend/` - серверная часть (Node.js/Express)
- `pages/` - страницы сайта (курсы, специалисты, супервизии)
- `includes/` - общие компоненты
- `images/` - медиафайлы

## Установка

1. Установите зависимости:
```bash
npm install
cd backend && npm install
```

2. Настройте переменные окружения в `.env` файле

3. Запустите сервер:
```bash
npm run start
```

## API

Backend API доступен на http://localhost:3001

### Основные эндпоинты:

#### Курсы и вебинары
- `GET /api/courses` - получить все курсы
- `GET /api/courses/:id` - получить курс по ID
- `GET /api/courses/slug/:slug` - получить курс по slug
- `POST /api/courses` - создать курс
- `PUT /api/courses/:id` - обновить курс
- `DELETE /api/courses/:id` - удалить курс

#### Специалисты
- `GET /api/specialists` - получить всех специалистов
- `GET /api/specialists/:id` - получить специалиста по ID
- `POST /api/specialists` - создать специалиста
- `PUT /api/specialists/:id` - обновить специалиста
- `DELETE /api/specialists/:id` - удалить специалиста

#### Заявки
- `GET /api/requests` - получить все заявки
- `POST /api/requests` - создать заявку
- `PUT /api/requests/:id` - обновить заявку
- `DELETE /api/requests/:id` - удалить заявку

#### Промокоды
- `GET /api/promo-codes` - получить все промокоды
- `POST /api/promo-codes` - создать промокод
- `POST /api/promo-codes/validate` - проверить промокод
