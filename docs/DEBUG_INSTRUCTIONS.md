# Инструкция по отладке

## Шаги для проверки:

### 1. Полностью очистить кэш браузера
- Откройте DevTools (F12)
- Перейдите на вкладку Network
- Поставьте галочку "Disable cache"
- Обновите страницу (Ctrl+F5 или Cmd+Shift+R)

### 2. Проверить консоль
После обновления страницы в консоли должно появиться:
```
admin-courses-certificates.js loaded
```

### 3. Кликнуть на вкладку "Курсы"
В консоли должно появиться:
```
loadCourses called
```

### 4. Если видите ошибку "loadCourses is not defined"
Это значит файл не загрузился. Проверьте:
- Открыт ли файл через http://localhost:8000 (НЕ file://)
- Загружается ли файл admin-courses-certificates.js?v=24
- Есть ли ошибки загрузки в Network tab

### 5. Альтернативный способ
Откройте консоль и введите:
```javascript
console.log(typeof window.loadCourses);
```

Должно вывести: `function`

Если выводит `undefined`, значит файл не загрузился.

## Текущая версия файлов: v=24

## URL для проверки:
```
http://localhost:8000/admin/pages/admin-panel.html
```

## Если ничего не помогает:

### Вариант 1: Закрыть все вкладки браузера
1. Закройте ВСЕ вкладки с админкой
2. Закройте браузер полностью
3. Откройте браузер заново
4. Откройте админку

### Вариант 2: Использовать режим инкогнито
1. Откройте браузер в режиме инкогнито (Ctrl+Shift+N)
2. Откройте админку
3. Проверьте работу

### Вариант 3: Другой браузер
Попробуйте открыть в другом браузере (Chrome, Firefox, Safari)
