# Исправление: Список курсов в админке

## Проблема:
Список курсов в админке был пустым, показывалось "Функционал в разработке..."

## Причина:
В `admin/js/admin-panel.js` была старая функция `loadCourses()` которая перекрывала новую функцию из `admin/js/admin-courses-certificates.js`

## Решение:
Удалена старая функция `loadCourses` из `admin-panel.js`

## Что было:
```javascript
// admin-panel.js
function loadCourses() {
    pageTitle.textContent = 'Курсы';
    pageSubtitle.textContent = 'Управление курсами и вебинарами';
    adminContent.innerHTML = `
        <p>Функционал в разработке...</p>
    `;
}
```

## Что стало:
```javascript
// admin-panel.js
// loadCourses is defined in admin-courses-certificates.js
```

Теперь используется правильная функция из `admin-courses-certificates.js` которая:
- Загружает курсы через API
- Отображает карточки курсов
- Показывает кнопки редактирования и удаления

## Результат:
✅ В админке теперь отображаются все 4 курса:
1. Марафон по РПП «Я больше не переедаю» - 9900 ₽
2. Работа в психосоматических запросах - 15900 ₽
3. Основы EMDR-терапии - 18900 ₽
4. Семейная психотерапия - 16900 ₽

## Как проверить:
1. Открыть админку: `http://localhost:3001/admin/pages/admin-panel.html`
2. Перейти в раздел "Курсы"
3. Увидеть список из 4 курсов с карточками
4. Можно редактировать и удалять курсы
