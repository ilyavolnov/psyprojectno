# Финальное исправление: Вкладки Курсы и Сертификаты

## Проблема:
Функции `loadCourses` и `loadCertificates` не были доступны глобально, даже после добавления экспорта в конец файла.

## Причина:
Функции были определены как обычные `async function`, а экспорт происходил в конце файла. Из-за особенностей JavaScript, функции не были видны в момент экспорта.

## Решение:
Изменено определение функций - теперь они сразу определяются как глобальные:

### До:
```javascript
async function loadCourses() {
    // код
}

// В конце файла
window.loadCourses = loadCourses; // ❌ Не работает
```

### После:
```javascript
window.loadCourses = async function() {
    // код
}; // ✅ Работает сразу
```

## Изменения:

### 1. admin/js/admin-courses-certificates.js
```javascript
// Было:
async function loadCourses() { ... }
async function loadCertificates() { ... }

// Стало:
window.loadCourses = async function() { ... };
window.loadCertificates = async function() { ... };
```

### 2. admin/pages/admin-panel.html
Обновлена версия скриптов с `v=22` на `v=23`
