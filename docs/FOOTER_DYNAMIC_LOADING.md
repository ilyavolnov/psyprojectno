# Динамическая загрузка футера

## Что реализовано

Создана система динамической загрузки футера для упрощения поддержки сайта. Теперь футер хранится в одном месте и автоматически загружается на страницы.

## Файлы

### 1. includes/footer.html
**Типовой шаблон футера** - единственный источник правды для футера сайта.

Содержит:
- ✅ Информацию о компании (ИП, ОГРНИП, адрес)
- ✅ Навигацию по сайту
- ✅ Блок услуг с WhatsApp кнопкой
- ✅ Социальные сети (VK, Instagram, Telegram, YouTube)
- ✅ Юридические ссылки
- ✅ Скрытую ссылку на админку

### 2. footer-loader.js
**Загрузчик футера** - JavaScript скрипт, который динамически загружает футер.

```javascript
// Автоматически загружает includes/footer.html
// Заменяет существующий <footer> или добавляет в конец body
```

## Использование на страницах

### Страницы с динамической загрузкой

#### index.html ✅
```html
<!-- Footer (loaded dynamically) -->
<footer class="footer"></footer>

<script src="footer-loader.js"></script>
<script src="smooth-scroll.js"></script>
<script src="script.js"></script>
```

#### course-page.html ✅
```html
<!-- Footer (loaded dynamically) -->
```

## Преимущества

✅ **Единый источник** - футер редактируется в одном месте
✅ **Автоматическое обновление** - изменения применяются ко всем страницам
✅ **Упрощение поддержки** - не нужно копировать код футера
✅ **Консистентность** - все страницы имеют одинаковый футер
✅ **Легкость обновления** - изменил один файл, обновились все страницы

## Как обновить футер

1. Откройте `includes/footer.html`
2. Внесите изменения
3. Сохраните файл
4. Обновите страницы в браузере (Ctrl+F5)
5. Все страницы с динамической загрузкой автоматически получат обновления

## Как добавить динамический футер на новую страницу

1. Замените весь блок `<footer>...</footer>` на:
```html
<!-- Footer (loaded dynamically) -->
<footer class="footer"></footer>
```

2. Добавьте скрипт перед закрывающим `</body>`:
```html
<script src="footer-loader.js"></script>
```

3. Готово! Футер загрузится автоматически.

## Технические детали

### Как работает footer-loader.js

1. Скрипт загружается при загрузке страницы
2. Делает fetch запрос к `includes/footer.html`
3. Получает HTML футера
4. Находит элемент `<footer class="footer">` на странице
5. Заменяет его содержимое на загруженный HTML
6. Если футера нет, добавляет в конец body

### Совместимость

- ✅ Работает во всех современных браузерах
- ✅ Не требует jQuery или других библиотек
- ✅ Асинхронная загрузка (не блокирует рендеринг)
- ✅ Обработка ошибок (если файл не найден)

### Производительность

- Минимальное влияние на скорость загрузки
- Кэшируется браузером
- Загружается параллельно с другими ресурсами

## Альтернативный подход

Если динамическая загрузка не подходит, можно:
1. Использовать SSI (Server Side Includes)
2. Использовать PHP include
3. Использовать систему сборки (Webpack, Gulp)
4. Оставить встроенный футер (как в certificates.html)

Текущий подход с JavaScript - самый простой и не требует серверной настройки.
